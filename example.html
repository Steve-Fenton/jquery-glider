FeatureCode
Basic Use 

<div class="slide">
    <ul>
        <li>One</li>
        <li>Two</li>
        <li>Three</li>
    </ul>
</div>
<script src="jquery.min.js"></script>
<script src="glider.js"></script>
<script>
    var gliders = $('.slide').glider();	
</script>
Right to Left dir="rtl"
Autoplay data-glider-autoplay or data-glider-autoplay="true"
Controls data-glider-controls="glider-top" with glider-top, glider-middle, or glider-bottom
Advance to next slide slider.next();
Return to previous slide slider.back();
Go to a specific slide slider.goto(0);
Get a control element (so it can be placed outside of the slider) 

var backControl = slider.getBackControl('class-name', 'Back');
var nextControl = slider.getNextControl('class-name', 'Next');
var linkControl = slider.getLinkControl('class-name', function (idx) { return idx + 1; });
